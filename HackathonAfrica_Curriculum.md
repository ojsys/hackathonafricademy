# HackathonAfrica LMS — Curriculum Outline

**Program:** Web Development Fundamentals  
**Delivered by:** AfricaPlan Foundation (africaplanfoundation.org)  
**Target Audience:** African youth with no prior coding experience  
**Total Duration:** 44 hours across 3 courses  
**Delivery Format:** Self-paced online learning with interactive exercises and assessments

---

## Program Overview

The HackathonAfrica curriculum is a beginner-friendly, three-course web development track designed to take students from zero coding experience to building real websites. Upon successful completion, students become eligible for HackathonAfrica hackathon opportunities.

### Eligibility Requirements
To qualify for HackathonAfrica programs, students must:
1. Complete all 3 courses
2. Pass all module quizzes (minimum score: 70%)
3. Pass all final exams (minimum score: 70%)
4. Receive admin approval (optional additional review)

### Assessment Structure
- **Module Quizzes:** 12 quizzes (one per module), 5 questions each, 70% minimum pass mark
- **Final Exams:** 3 exams (one per course), MCQ + coding exercises, 70% minimum pass mark
- **Interactive Exercises:** Hands-on coding exercises embedded throughout each lesson
- **Live Preview:** Students see their code rendered in real time

### Learning Outcomes
By completing this curriculum, students will be able to:
- Build structured, semantic web pages using HTML5
- Style and layout pages with modern CSS including Flexbox and Grid
- Add interactivity with JavaScript including DOM manipulation and async data fetching
- Apply web accessibility and performance best practices
- Work confidently in a real development environment
